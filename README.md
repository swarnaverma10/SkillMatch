# 🤖 Generative AI Powered Job Recommender System

An intelligent AI-based job recommender system that analyzes resumes and suggests relevant job roles using NLP and transformer-based models.

---

## 🚀 Features

- 📄 Resume PDF text extraction
- 🧠 AI-powered resume summarization
- 🔍 Skill gap & missing area analysis
- 🏷️ Job keyword extraction
- 💼 Job recommendations from LinkedIn & Naukri
- ⚡ Fast local inference using Hugging Face models

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Hugging Face Transformers
- NLP
- PyMuPDF (PDF parsing)
- Apify API
